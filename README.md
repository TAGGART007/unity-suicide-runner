# Suicide Runner - Unity 2D Runner & Shooter

**Suicide Runner** is a 2D endless runner game built in **Unity 2D**. The player controls a character that runs forward automatically, jumps over obstacles, collects coins, and shoots enemies.  

This game combines **endless runner mechanics with shooting gameplay** for more action and challenge.

---

## Gameplay Features

- **Character Movement**
  - Runs automatically along the path
  - Jump over obstacles using **Spacebar** (keyboard) or touch (mobile)

- **Shooting Mechanics**
  - Shoot enemies or targets while running
  - Use **Left Click** (mouse) or touch button for shooting

- **Collectibles**
  - Coins scattered along the path increase the score
  - Bonus items or power-ups can appear

- **Obstacles**
  - Avoid obstacles by jumping or shooting
  - Colliding with obstacles ends the game

- **Scoring**
  - Score increases over distance and by collecting coins
  - Tracks **high score** for replay motivation

- **Endless Gameplay**
  - Game continues until the player dies

---

## Tech Stack

- **Engine:** Unity 2D (2019/2020/2021 or later)  
- **Language:** C#  
- **Assets:** 2D sprites for player, obstacles, coins, and enemies  
- **Input Controls:** Keyboard + Mouse or Mobile Touch
